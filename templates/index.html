DS 440 Capstone

A personal music analytics and recommendation platform powered by your Last.fm listening history.

Listening History

Automatically syncs your scrobbles from Last.fm. Browse your recent plays, top artists, tracks, and albums across different time periods.

Music Discovery

Get personalized recommendations using a hybrid tag-based engine. Choose Comfort Zone to find more of what you love, or Branch Out to explore new sounds.

How It Works

The engine analyzes your listening patterns, artist tags, and co-listening habits to build a taste profile, then scores candidate tracks against it.

Tag-Based Matching

Artist and track tags from Last.fm are compared using cosine similarity. The more complete your tag data, the better the recommendations.

Getting Started

  1. Connect your Last.fm account via the Settings button above
  2. Your listening history will sync automatically in the background
  3. Tag and similar artist data is collected gradually to power recommendations
  4. Head to Discover to generate personalized recommendations
  5. Give feedback on recommendations to improve future suggestions

Setup Guide

Follow these steps to run the application locally on your machine.

Prerequisites

  • Python 3.9+ — Download here
  • Last.fm Account — Create one here
  • Last.fm API Key — Get your API key

Installation

1. Clone the Repository

git clone https://github.com/YOUR_USERNAME/lastfm-tracker.git
cd lastfm-tracker

2. Create Virtual Environment

python -m venv venv

# Windows
venv\Scripts\activate

# macOS/Linux
source venv/bin/activate

3. Install Dependencies

pip install -r requirements.txt

4. Run the Application

python app.py

The app will start at http://127.0.0.1:5000

Configuration

  1. Open http://127.0.0.1:5000 in your browser
  2. Click the Settings button in the header
  3. Enter your Last.fm Username and API Key
  4. Click Save & Connect
  5. Your listening history will begin syncing automatically

How to Get Your Last.fm API Key

  1. Go to Last.fm API Account Creation
  2. Log in with your Last.fm account
  3. Fill in any application name and description
  4. Leave Callback URL blank
  5. Submit and copy your API Key (you don't need the shared secret)

Troubleshooting

"Invalid API key" — Double-check your API key has no extra spaces.

Sync not running — Make sure the terminal running python app.py stays open.

Missing scrobbles — Last.fm may have delays. Wait a few minutes and sync again.
