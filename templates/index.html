DS 440 Capstone

A personal music analytics and recommendation platform powered by your Last.fm listening history.

📊

Listening History

Automatically syncs your scrobbles from Last.fm. Browse your recent plays, top artists, tracks, and albums across different time periods.

🎵

Music Discovery

Get personalized recommendations using a hybrid tag-based engine. Choose Comfort Zone to find more of what you love, or Branch Out to explore new sounds.

🧠

How It Works

The recommendation engine analyzes your listening patterns, artist tags, and co-listening habits to build a profile of your taste, then scores candidate tracks against it.

🔍

Tag-Based Matching

Artist and track tags from Last.fm are compared using cosine similarity. The more complete your tag data, the better the recommendations become.

Getting Started

  1. Connect your Last.fm account via the settings button above
  2. Your listening history will sync automatically in the background
  3. Tag and similar artist data is collected gradually to power recommendations
  4. Head to the Discover tab to generate personalized recommendations
  5. Give feedback on recommendations to improve future suggestions

Setup Guide

Follow these steps to run the application locally on your machine.

Prerequisites

  • Python 3.9+ - Download here
  • Last.fm Account - Create one here
  • Last.fm API Key - Get your API key

Installation

1. Clone the Repository

git clone https://github.com/YOUR_USERNAME/lastfm-tracker.git
cd lastfm-tracker

2. Create Virtual Environment

# Create virtual environment
python -m venv venv

# Activate it (Windows)
venv\Scripts\activate

# Activate it (macOS/Linux)
source venv/bin/activate

3. Install Dependencies

pip install -r requirements.txt

4. Run the Application

python app.py

The app will start at http://localhost:5000

Configuration

  1. Open http://localhost:5000 in your browser
  2. Click the Settings button (gear icon) in the header
  3. Enter your Last.fm Username
  4. Enter your Last.fm API Key
  5. Click Save & Connect
  6. Your listening history will begin syncing automatically

How to Get Your Last.fm API Key

  1. Go to Last.fm API Account Creation
  2. Log in with your Last.fm account
  3. Fill in the application details:
    • Application name: "Personal Listening Tracker"
    • Application description: "Personal music analytics"
    • Callback URL: Leave blank
  4. Click Submit
  5. Copy your API Key (you don't need the shared secret)

Troubleshooting

"Invalid API key" error: Double-check your API key has no extra spaces.

Sync not running: Make sure the terminal running python app.py stays open.

Missing scrobbles: Last.fm may have delays. Wait a few minutes and try syncing again.
